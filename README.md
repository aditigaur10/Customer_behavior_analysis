Retail Consumer Behavior Analysis
Overview

This project analyzes a retail consumer behavior dataset to uncover purchasing trends, customer preferences, and key sales drivers. The goal is to generate actionable insights that improve customer engagement, optimize marketing strategies, and support data-driven business decisions.

Dataset

The dataset includes customer demographics, product categories, sales channels (online/offline), discounts, reviews, seasonal trends, payment methods, and purchase history.

It is used to:

Analyze buying patterns across segments

Identify factors influencing purchase and repeat behavior

Compare performance across channels and categories

Tools & Technologies

Python (Pandas, NumPy, Matplotlib/Seaborn) – Data cleaning & EDA

SQL (PostgreSQL / MySQL / SQL Server) – Data querying & analysis

Power BI – Interactive dashboard creation

Microsoft Excel / PowerPoint – Reporting

Gamma – Presentation design
